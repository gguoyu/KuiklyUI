### Kuikly DSL API 知识积累

- **Canvas API**: `Canvas({init}, drawCallback)` — drawCallback: `(context: CanvasContext, width: Float, height: Float) -> Unit`
  - 路径: `beginPath()`, `moveTo()`, `lineTo()`, `arc()`, `quadraticCurveTo()`, `bezierCurveTo()`, `closePath()`

  - 渐变: `createLinearGradient(x0, y0, x1, y1)` → `.addColorStop(position, color)`
  - 文本: `font(size)`, `fillText(text, x, y)`, `strokeText(text, x, y)`, `textAlign(TextAlign.LEFT/CENTER/RIGHT)`
  - 状态: `save()`, `restore()`, `translate()`, `scale()`, `rotate()`

- **GradientRichText 原理**: 不是独立组件，而是 `Text`/`RichText` 调用 `backgroundLinearGradient()` 后 `viewName()` 自动返回 `"KRGradientRichTextView"`
  - Web 实现: CSS `background-image: linear-gradient(...) + background-clip: text + color: transparent`

- **Scroller API**: `Scroller { attr { flex(1f); scrollEnable(true); flexDirectionRow() // 水平 } }`
  - 属性: `scrollEnable()`, `bouncesEnable()`, `showScrollerIndicator()`, `pagingEnable()`
  - 内部自动创建 `KRScrollContentView` 容器

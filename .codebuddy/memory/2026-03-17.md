### Kuikly DSL 样式 API 知识积累
- `border(Border(width, BorderStyle.SOLID/DASHED/DOTTED, Color))` — 边框
- `borderRadius(f)` / `borderRadius(BorderRectRadius(...))` — 圆角
- `boxShadow(BoxShadow(offsetX, offsetY, shadowRadius, shadowColor))` — 阴影
- `backgroundLinearGradient(Direction.TO_RIGHT, ColorStop(...), ...)` — 渐变
- `transform(rotate=Rotate(), scale=Scale(), translate=Translate(), skew=Skew(), anchor=Anchor())` — 变换
- `opacity(0.0f ~ 1.0f)` — 透明度
- `overflow(clipChild: Boolean)` — true=允许溢出, false=裁剪
